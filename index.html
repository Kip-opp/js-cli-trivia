About This Project

This is a CLI (Command Line Interface) application that tests your knowledge of JavaScript concepts. It features:

  • Timed Gameplay: 2-minute strict time limit.
  • Score Tracking: Immediate feedback and final scoring.
  • Tech Stack: Node.js, Readline, JavaScript (ES6+).
